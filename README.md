# ✨ Aqiqah Invitation — Sariym Hasan

A beautiful, interactive digital Aqiqah invitation for **Sariym Hasan**. Built with pure HTML, CSS, and vanilla JavaScript — no frameworks, no build tools.

## 🌙 Features

- **Elegant veil opening animation** with Arabic Bismillah calligraphy
- **Scratch-to-reveal** interactive date reveal
- **Live countdown timer** to the celebration
- **Photo gallery** with hover effects
- **Floating golden sparkle** particle effects
- **Scroll-based animations** for smooth section reveals
- **Background audio** toggle (Nasheed)
- **Google Maps** venue link
- **Fully responsive** — works on mobile, tablet, and desktop

## 🎨 Design

- **Color palette**: Gold (#C99E32) & Ivory (#FAF6F0)
- **Typography**: Cormorant Garamond, Marcellus, Inter (Google Fonts)
- **Style**: Elegant, minimal Islamic design with geometric accents

## 🛠️ Tech Stack

- HTML5
- CSS3 (custom properties, animations, clamp)
- Vanilla JavaScript (IntersectionObserver, Canvas API)
- Google Fonts

## 🚀 Getting Started

### Run Locally

Simply open the HTML file in any modern browser:

```bash
# No build step needed — just open the file
open "aqiqah-invitation-sariym-hasan trial 1.html"
```

Or use a local server:

```bash
# Using Python
python -m http.server 3000

# Using Node.js (npx)
npx serve .
```

Then visit `http://localhost:3000`

### Deploy on Vercel

1. Push this repository to GitHub
2. Go to [vercel.com](https://vercel.com) and import the repository
3. Vercel will auto-detect it as a static site
4. Click **Deploy** — done!

Alternatively, deploy via CLI:

```bash
npm i -g vercel
vercel
```
